## Featured Project

### Adaptive Financial Security Framework — Smart QR ATM System

> *Reimagining ATM authentication for the post-PIN era.*

**The Problem**

Traditional ATM PIN systems are a 40-year-old security model. They're vulnerable to shoulder surfing, card skimming, and social engineering. In a world where your phone authenticates your banking app with a QR code, why is your ATM still asking for 4 digits?

**The Solution**

A secure, QR-based ATM authentication system that eliminates static PINs entirely. Users authenticate via a dynamically generated QR code — invalidated after each session — with a time-limited fallback PIN for edge cases.

**Key Features**

- Dynamic QR code generation tied to session context — no reuse possible
- Temporary PIN fallback with expiry window for accessibility
- Image processing pipeline for reliable QR decoding under real-world conditions
- Session invalidation on timeout or failed attempt threshold

**Impact**

- Eliminates the primary attack vector for PIN-based card fraud
- Reduces authentication time by removing manual digit entry
- Designed to be drop-in compatible with existing ATM hardware flow
- Proof-of-concept validated against standard QR decode accuracy benchmarks

**Stack:** Python · OpenCV · Image Processing · QR Generation

### Photography Portfolio — Client-Ready Web Presence

A full-featured, conversion-optimised portfolio site built for a real photography business use case. Designed with the end client in mind — not just the code reviewer.

**What makes it notable**

This isn't a template clone. Every interaction — the lightbox, the mode toggle, the contact flow — was designed to reduce friction between a potential client and a booking. That's product thinking, not just frontend development.

**Highlights**

- Persistent dark/light mode toggle with `localStorage` — preference survives page refresh
- Custom image lightbox with keyboard navigation (no third-party lightbox library)
- Scroll-triggered animations using Intersection Observer API for performance
- WhatsApp deeplink integration for zero-friction mobile contact
- Sub-3-second load time through manual asset optimisation
- Fully responsive across all viewport sizes

**Stack:** HTML5 · CSS3 · JavaScript (ES6+) · Intersection Observer API
